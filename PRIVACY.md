# RypeBot Privacy Policy

**Last Updated:** July 5, 2026

## Overview

RypeBot ("the Bot") is a Discord application developed to provide server management, utility, and League of Legends related features.

Your privacy is important. This Privacy Policy explains what information RypeBot collects, how it is used, and your rights regarding that information.

By using RypeBot, you agree to the practices described in this Privacy Policy.

---

# Information We Collect

Depending on which commands are used, RypeBot may collect or process:

- Discord User IDs
- Discord Server (Guild) IDs
- Discord Channel IDs
- Discord Role IDs
- Command interaction data
- Server configuration settings
- Temporary cached API responses
- Error and diagnostic logs

RypeBot does **not** intentionally collect:

- Discord passwords
- Discord authentication tokens
- Email addresses
- Payment information
- Private messages unless explicitly required by a feature
- Personal information beyond what Discord provides through its API

---

# How Information Is Used

Collected information is used solely to:

- Execute Discord commands
- Provide bot functionality
- Store server-specific settings
- Verify permissions
- Improve performance
- Troubleshoot errors
- Prevent abuse
- Maintain service reliability

---

# Third-Party Services

RypeBot may communicate with third-party services, including but not limited to:

- Discord API
- Riot Games API
- Riot Data Dragon
- Riot Community Dragon
- League of Legends Wiki
- GitHub

These services maintain their own privacy policies.

---

# Data Storage

RypeBot may store limited information necessary for operation, including:

- Server configuration
- Feature settings
- Cached responses
- Temporary operational data

Data is retained only as long as reasonably necessary to provide functionality.

---

# Data Sharing

RypeBot does not sell, rent, or trade your personal information.

Information may only be disclosed:

- When required by law
- To comply with legal obligations
- To protect the integrity and security of the service

---

# Data Security

Reasonable administrative and technical measures are used to protect stored information. However, no system can guarantee absolute security.

---

# Your Rights

Server owners may request removal of server-specific configuration data.

Users may request deletion of stored user-specific information where applicable.

---

# Children's Privacy

RypeBot is not directed toward children under the age required to use Discord under Discord's Terms of Service.
